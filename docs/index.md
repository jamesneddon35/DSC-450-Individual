# James Sneddon

**Data scientist & goalkeeper.** B.S. Data Science (in progress), Bellevue
University, with professional soccer experience in USL League One. I build
models where football and data meet — plus NLP and applied-regression work
from outside the sport.

---

## [USL Match Points Predictor](usl-match-points.md)

An end-to-end machine learning project predicting match points for USL
Championship 2025 teams — from exploratory analysis through feature
engineering to a tuned gradient-boosted model served in an interactive Dash
app. Goalkeeper value enters through American Soccer Analysis **goals-added
(g+)** metrics.

## [IMDB 50K Sentiment Analysis](imdb-sentiment.md)

A two-stage NLP pipeline on the classic IMDB 50K reviews dataset: rigorous
text normalization and TF-IDF/Word2Vec feature engineering, then Logistic
Regression and Linear SVC classifiers. Includes a CRF named-entity tagger
side project.

## [Almond Chill Accumulation vs. Yield](almond-chill.md)

Does winter chill drive almond yield? USDA and NOAA data wrangling, chill-hour
feature engineering, and OLS regression for Fresno County — with an honest
treatment of confounds, small samples, and ethics.
